This session will ask students to try out games that they normally wouldn't and report back on the experience.

Learning Objectives:
  • Find games to try that you wouldn't normally play
  • Report on the experience and mechanics of the game
  • Suggest ways you would have improved the game
Agenda:
0000 - 0010 Arrival and Registration
0010 - 0140 Find, play, and report about a game
0140 - 0200 Coffee Break
0200 - 0245 General Programming Support and Discussion
0245 - 0250 Debrief
0250 - 0300 Departure

Further Guidance:
  • Find and play a game you usually wouldn't. Try the free games section on itch.io or check out some browser based games.
  • Write a short essay about the experience and mechanics of the game, include how you would like to see it improved.
  • Fork the Programmer's Guild Repository
  • Upload your essay and create a pull request

  • Try out other games! Theres a lot to learn from seeing other peoples work that you usually wouldn't
